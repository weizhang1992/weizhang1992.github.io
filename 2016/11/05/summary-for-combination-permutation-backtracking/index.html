basic knowledge:

  1. Combination:

    • choose k from n ignore order, represented as C(n, k)
    • C(n, k) = n! / (k! * (n - k)!)
    • C(n, k) = C(n - 1, k - 1) + C(n - 1, k)
    • C(n, 0) + C(n, 1) + C(n, 2) + … + C(n, n) = 2 ^ n
    • use above equation to do time complexity analysis
